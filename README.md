# fis-cross-app


## Features
- Can reload the fis app when the referred app changed

## Install
npm install fis-cross-app -g

## How to Use

### Import the module
```js
var fisCrossApp = require('fis-cross-app');
```
### Set project name as a param
```js
fis.config.set('usedByNS', 'home');
```
### That's all, enjoy it
